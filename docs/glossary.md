# Glossary

- Block: A batch of transactions added to the chain.
- Transaction (tx): A record of value or data sent on-chain.
- Address: An account identifier (like a public mailbox).
- Native token: The main coin of the chain (ETH on Ethereum).
- Transfer: Moving value from one address to another.
- RPC (JSON-RPC): A way to ask a node for blockchain data.
- ETL: Extract, Transform, Load (move data into a database).
- Ingestion: The process of collecting and saving data.
- Anomaly: Something unusual compared to normal behavior.
- Whale: A very large holder that moves big amounts.
